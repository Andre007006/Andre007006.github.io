Pendidikan Saya Saat Ini Di SMK

Dipublikasikan pada: 14 Desember 2024

Setelah Lulus dari SMP saya bingung untuk menentukan sekolah SMA atau SMK, Namun tentu saya harus berpikir lagi sebelum melanjutkan sekolah. Setelah saya berpikir, saya menentukan pilihan saya yaitu SMK, ada banyak pilihan sekolah SMK, saya memilih di SMK Negeri Tambakboyo.

Sekolah SMk Negeri Tambakboyo memiliki 7 jurusan yaitu:

  1. Teknik Pemesinan (TPM)
  2. Teknik Kendaraan Ringan (TKR)
  3. Teknik Komputer dan Jaringan (TKJ)
  4. Rekaya Perangkat Lunak (RPL)
  5. Teknik Pemesinan Kapal (TPK)
  6. Teknik Instalasi Tenaga listrik (TITL)
  7. Tata Busana (TB)
Saya memilih jurusan Rekaya Perangkat Lunak (RPL), Ini awal tantangan saya untuk menjadi web developer.

Menjadi web developer bukan tantangan yang mudah, harus memiliki

Tantangan dan Peluang
Belajar menjadi web developer bukanlah perjalanan yang mudah, tetapi menawarkan banyak peluang menarik. Dalam era digital saat ini, keterampilan dalam pengembangan web sangat dicari. Namun, banyak tantangan yang harus dihadapi oleh mereka yang ingin memasuki bidang ini. Di sisi lain, peluang karier di bidang web development sangat luas. Permintaan untuk web developer terus meningkat seiring dengan pertumbuhan bisnis online. Banyak perusahaan mencari profesional yang mampu menciptakan situs web yang menarik dan fungsional. Selain itu, ada kemungkinan untuk bekerja secara freelance, memberikan fleksibilitas dalam waktu dan lokasi kerja. Akhir Kata: Tetap semangat berproses karena sukses tanpa proses hanya mimpi
